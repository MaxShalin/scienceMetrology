Описание проекта.

Среда разработки: python 3.9

Структура:
Фаил "Оценка.exe" - скомпелированная программа;
Фаил "db_measure.db" - база данных SQLite;
Фаил "requirements.txt" - описание зависимостей;
Фаилы "*.py" - модули python;
Папка qt_forms - PtQt5 файлы для интерфейса программы 

Для понимания необходимо знать:
1. Язык програмирования python. 
   Рекомендуемый ресурс - https://www.youtube.com/watch?v=5g-MHZ0MzZY.

2. Систему управления БД SQLite.
   Рекомендуемый ресурс - https://www.youtube.com/watch?v=mRuylFh9wfY.

3. Библиотеку PyQt5 для создания графического интерфейса на базе платформы Qt5.
   Рекомендуемый ресурс - https://pythonworld.ru/gui/pyqt5-firstprograms.html.

4. Qt Designer - инструмент Qt для проектирования и создания графических пользовательских интерфейсов (GUI).
   Рекомендуемый ресурс - https://doc.qt.io/qt-5/qtdesigner-manual.html.
